# Beginning Level Data Science Projects

This repository contains beginner-friendly data science and machine learning projects.

## Projects Included

### 1. BMI Calculator & Analysis
- **Location**: `BMI4/`
- **Description**: BMI calculation with health analysis
- **Technologies**: Python, Streamlit

### 2. COVID-19 Data Analysis
- **Location**: `Co_19/`
- **Description**: Global COVID-19 trends and visualization
- **Technologies**: Python, Pandas, Matplotlib

### 3. Diabetes Prediction
- **Location**: `Diabetes/`
- **Description**: Machine learning model for diabetes prediction
- **Technologies**: Python, Scikit-learn

### 4. Exploratory Data Analysis (EDA)
- **Location**: `EDA/`
- **Description**: Comprehensive EDA on Titanic dataset
- **Technologies**: Python, Pandas, Seaborn

### 5. Web Scraper
- **Location**: `Scraper/`
- **Description**: Web scraping tool for data collection
- **Technologies**: Python, BeautifulSoup

### 6. Temperature Time Series Analysis
- **Location**: `Temperature/`
- **Description**: Time series analysis of temperature data
- **Technologies**: Python, Time Series Analysis

## Getting Started

1. Clone the repository
2. Navigate to any project folder
3. Install requirements: `pip install -r requirements.txt`
4. Run the project as per individual README files
