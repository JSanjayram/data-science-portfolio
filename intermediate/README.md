# Intermediate Level Data Science Projects

This repository contains intermediate-level data science and machine learning projects with more complex algorithms and implementations.

## Projects Included

### 1. Car Price Prediction
- **Location**: `car_pricec/`
- **Description**: ML model to predict car prices with feature engineering
- **Technologies**: Python, Scikit-learn, Streamlit
- **Features**: Model persistence, preprocessing pipelines

### 2. Customer Segmentation Analysis
- **Location**: `customer/`
- **Description**: Advanced customer segmentation using clustering algorithms
- **Technologies**: Python, K-Means, Data Visualization
- **Features**: Interactive dashboard, segment analysis

### 3. Employee Attrition Prediction
- **Location**: `Employee_Attrition_Prediction/`
- **Description**: Predict employee turnover using HR data
- **Technologies**: Python, Classification algorithms
- **Features**: Feature importance analysis, model evaluation

### 4. Student Performance Prediction
- **Location**: `Student Performance Prediction/`
- **Description**: Predict student academic performance
- **Technologies**: Python, Regression models, Feature engineering
- **Features**: Model comparison, performance metrics

### 5. Student Performance Analysis
- **Location**: `Students/`
- **Description**: Comprehensive analysis of student performance factors
- **Technologies**: Python, Statistical analysis, Correlation analysis
- **Features**: Advanced visualizations, statistical insights

## Skills Demonstrated

- Advanced Machine Learning
- Feature Engineering
- Model Persistence
- Data Preprocessing Pipelines
- Customer Analytics
- HR Analytics
- Educational Data Mining
- Interactive Dashboards
- Statistical Analysis

## Getting Started

1. Clone the repository
2. Navigate to any project folder
3. Install requirements: `pip install -r requirements.txt`
4. Run the project as per individual README files
