# Advanced Level Data Science Projects

This repository contains advanced-level data science, machine learning, and deep learning projects with complex implementations and production-ready features.

## Projects Included

### 1. Cat vs Dog Classification
- **Location**: `cat vs dog/`
- **Description**: Deep learning image classification using CNN
- **Technologies**: TensorFlow/Keras, Computer Vision, Streamlit
- **Features**: Real-time prediction, model deployment

### 2. Employee Attrition Prediction (Advanced)
- **Location**: `Employee Attrition Prediction/`
- **Description**: Comprehensive HR analytics with multiple ML models
- **Technologies**: Python, Advanced ML, Feature Engineering, Streamlit
- **Features**: Model comparison, hyperparameter tuning, detailed reporting

### 3. Face Mask Detection
- **Location**: `Face_Mask_Detection/`
- **Description**: Real-time face mask detection using computer vision
- **Technologies**: OpenCV, Deep Learning, Transfer Learning
- **Features**: Live webcam detection, Docker deployment

### 4. Food Image Classification
- **Location**: `Food_Image_Classification/`
- **Description**: Multi-class food image classification system
- **Technologies**: CNN, Transfer Learning, Data Augmentation
- **Features**: Pre-trained models, custom training pipeline

### 5. Intelligent Customer Assistant
- **Location**: `intelligent_customer_assistant/`
- **Description**: AI-powered customer service chatbot with NLP
- **Technologies**: NLP, Database Integration, Docker, API
- **Features**: Intent recognition, database queries, containerized deployment

### 6. LSTM Stock Prediction
- **Location**: `LSTM_Stock_Prediction/`
- **Description**: Time series forecasting for stock prices using LSTM
- **Technologies**: LSTM, Time Series Analysis, Deep Learning
- **Features**: Multi-step prediction, technical indicators

### 7. Stock Price Prediction with LSTM
- **Location**: `Stock_Price_Prediction_LSTM/`
- **Description**: Advanced stock market prediction system
- **Technologies**: Deep Learning, Financial Analysis, LSTM Networks
- **Features**: Real-time data processing, advanced metrics

## Getting Started

1. Clone the repository
2. Navigate to any project folder
3. Install requirements: `pip install -r requirements.txt`
4. Follow individual project setup instructions
5. For Docker projects: `docker-compose up`
